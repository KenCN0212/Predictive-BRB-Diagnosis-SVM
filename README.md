# Predictive-BRB-Diagnosis-SVM

Sistema académico de diagnóstico predictivo de fallas de barras rotas (BRB) en motores trifásicos de inducción mediante Máquinas de Soporte Vectorial (SVM) y predicción térmica utilizando SVR.

---

## Descripción

Este proyecto fue desarrollado como una propuesta académica orientada al mantenimiento predictivo de motores de inducción mediante técnicas de inteligencia artificial y análisis de variables térmicas, eléctricas y mecánicas.

La implementación integra:

- Clasificación de estados operativos mediante SVM.
- Predicción térmica del rotor utilizando SVR.
- Generación automática de métricas y figuras.
- Simulación de variables representativas del comportamiento físico del motor.
- Arquitectura conceptual para aplicaciones industriales futuras.

---

## Tecnologías utilizadas

- Matplotlib
- Scikit-Learn
